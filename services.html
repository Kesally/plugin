安装枫叶插件

枫叶插件安装教程

注意：枫叶插件只支持Yunzai-bot3.0

将hs-qiqi-plugin下载下来放在Yunzai-Bot/plugins目录下，重启Yunzai-Bot后即可使用。

推荐使用git进行安装，以方便后续升级。在Yunzai目录打开终端，运行以下指令

cd Yunzai-Bot

git clone https://gitee.com/kesally/hs-qiqi-cv-plugin.git ./plugins/hs-qiqi-plugin

或是

cd Yunzai-Bot

git clone https://github.com/Kesally/hs-qiqi-cv-plugin.git ./plugins/hs-qiqi-plugin

自助问题解决区

免责声明

功能仅限内部交流与小范围使用，请勿将Yunzai-Bot及hs-qiqi-plugin用于任何以盈利为目的的场景

图片与其他素材均来自于网络，仅供交流学习使用，如有侵权请联系，会立即删除.

语音无法发送问题

1.如果你的机器人后台反馈[无法转换，请确认你的FFmpeg是否可以处理此转换]就说明你的FFmpeg没有装好，重新安装FFmpeg后重启机器人即可

2.如果你的机器人后台并未反馈无法转换而是显示[ERROR]或是[ERR]的红色前缀报错，请尝试联系我们解决问题。

插件冲突问题

1.枫叶插件可能会与榴莲插件产生冲突，如有发生冲突问题，删除任意一个插件后重启机器人即可

2.枫叶插件只支持Yunzai-3.0机器人，不支持Yunzai2.0机器人，如有问题请检查机器人版本

枫叶插件注意事项

1.开启涩涩功能可能导致机器人账号封停，请谨慎开启。

2.电脑端和部分版本手机端QQ，Tim是无法收听高清语音的，请谨慎开启

3.如果你的枫叶更新插件报错了，可以尝试对机器人发送指令 #枫叶强制更新 以更新插件。注意：此指令使覆盖更新哦。
